# JavaScript Array Manipulation and User Info

## Student ID: [11020737]

## Description

This project demonstrates basic array manipulations and user information handling in JavaScript. It consists of three main tasks:

### Task 1: Array Manipulation
- **File:** `arrayManipulation.js`
- **Function:** `processArray`
- **Description:** Takes an array of numbers as input and returns a new array where each even number is squared and each odd number is tripled.

### Task 2: Array String Formatting
- **File:** `arrayManipulation.js`
- **Function:** `formatArrayStrings`
- **Description:** Takes two arrays as input: an array of strings and an array of numbers processed by `processArray`. Modifies each string based on its corresponding number: capitalize the entire string if the number is even, and convert the string to lowercase if the number is odd.

### Task 3: User Information
- **File:** `userInfo.js`
- **Function:** `createUserProfiles`
- **Description:** Takes an array of names and an array of modified names from Task 2, and returns an array of objects. Each object contains `originalName`, `modifiedName`, and an `id:11020737` (auto-incremented starting from 1).
